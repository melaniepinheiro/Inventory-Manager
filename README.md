# Inventory-Manager
Restaurant Inventory Management System - Phase 1

How to run:
1. Open the project folder in IntelliJ.
2. Make sure the three Java files are inside the src folder.
3. Run InventoryApplication.java.
4. Use inventory.txt when the program asks for a file name.

This phase uses:
- Java objects
- ArrayList
- Command line menu
- Text file loading and saving
- CRUD features
- Custom calculation for total inventory value

No GUI or database is included in Phase 1 because those are later phases.
