# Restaurant Inventory Management System - Phase 1

This project is a Restaurant Inventory Management System created in Java. It allows users to manage inventory using a command-line menu.

## Features

- Load inventory from a text file
- Display inventory items
- Add new inventory items
- Update inventory items
- Remove inventory items
- Save inventory to a text file
- Calculate the total inventory value
- Input validation to prevent invalid data and program crashes

## How to Run

1. Open the project in IntelliJ IDEA.
2. Make sure the Java files are inside the `src` folder.
3. Make sure `inventory.txt` is in the project folder.
4. Run `InventoryApplication.java`.
5. When prompted, enter `inventory.txt` to load the inventory.

## Technologies Used

- Java
- ArrayList
- Object-Oriented Programming
- File I/O

This project currently uses a command-line interface. A GUI and database will be added in later phases.
